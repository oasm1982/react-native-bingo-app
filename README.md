# React Native Bingo App

A mobile Bingo game built with React Native and Expo. Includes navigation between screens using React Navigation.

## Learning Goals

This project is part of my journey to become a **Software Tester**. I use it to practice:

- Understanding mobile app structure to plan test scenarios
- Identifying screens and navigation flows that need to be tested
- Learning the difference between mobile testing and web testing
- Exploring tools like **Detox** or **Jest + React Native Testing Library** for mobile test automation

## Tech Stack

- React Native 0.64
- Expo ~43
- React Navigation 6

## Running the App

```bash
yarn install
expo start
```

Then scan the QR code with the Expo Go app on your phone.
